# Explore Data

In this quick tutorial, you will explore some data resources to get yourself familiar with the NDP catalog and the different search functionalities 

## Substring Search

1 - Go the [NDP Catalog](https://nationaldataplatform.org/ckandata).

2 - Type `HPWREN Weather`. You will see a dataset called *HPWREN Weather Station Measurements* listed first. This is a dataset that we use as part of the [*Set up Workspace*](../quick-start/set-up-workspace.md) tutorial.

3 - To learn more about the dataset, click on *View More*. This will open a new tab were you will be able to explore the dataset metadata, as well as each of the resources that are part of the dataset.

<img src="../images/hpwren.png" style="border: 2px solid black;">

4 - To make your substring search more precise, you can select an organization and this will filter datasets that are exclusive to that organization

<img src="../images/org.png" style="border: 2px solid black;">

## Map Search

1 - Click on *Map Search*

2 - Type the example *lidar*. You will be able to see the sites where lidar data is available. 

<img src="../images/lidar-map.png" style="border: 2px solid black;">

3 - You can also draw a polygon, and obtain datasets for your area of interest.

<img src="../images/rectangle-lidar.png" style="border: 2px solid black;">

4 - Click on one of the yellow or green circles, and you will see all the overlapping datasets within that smaller polygon. 

<img src="../images/circle-lidar.png" style="border: 2px solid black;">

The blue dots represent the exact location of a particular dataset. Click a blue dot, or a resource from a list of overlapping datasets and you will see more information about such dataset.
